# fast_login_db frontend

## Install required dependencies
```
brew install node
```

```
npm install -g @vue/cli
```

```
vue create vue-project
```

```
npm install axios
```

```
node -v && npm -v
```

### Compiles and hot-reloads for development
```
npm run serve
```
